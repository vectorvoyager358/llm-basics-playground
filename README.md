# LLM Basics Playground

A Python-based LLM application for experimenting with AI tasks such as summarization, question answering, and content generation.

---

## Features

- **Text Summarization:** Summarize any long paragraph into concise bullet points.  
- **Q&A Bot:** Ask any question and get clear AI-generated answers.  
- **Content Generator:** Generate engaging content such as LinkedIn posts.

---

## Tech Stack

- Python  
- Gemini API (`google-genai`)  
- dotenv  
- Modular Python code for easy extension
